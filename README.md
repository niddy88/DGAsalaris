# DGAsalaris

Deze repository bevat een Streamlit-app die helpt berekenen hoeveel netto je uit je BV haalt in 2026 bij een combinatie van salaris en dividend.

Deployen op Streamlit Cloud:
1. Ga naar https://share.streamlit.io en klik "New app".
2. Koppel je GitHub repository niddy88/DGAsalaris.
3. Als main file path voer in: `streamlit_app.py` (of selecteer het bestand).
4. Streamlit installeert dependencies uit `requirements.txt` automatisch.

Lokaal testen:

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
